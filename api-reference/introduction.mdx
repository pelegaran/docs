## Overview

The API provides a way for developers to programmatically work with the same data available in the AI Visibility Dashboard. All endpoints return data in JSON format.

## Getting Started

To use the API, you'll need to get an API key to your Account.
For more details on how authentication works, see the [Authentication](/api-reference/authentication) section.
