## Quick reference

- **Base URL:** `https://api.mindshare.so`
- **Auth:** API key in header `Authorization: Bearer <your-api-key>`
- **Content type:** `application/json`

For full details, use the **Mindshare API** tab: [Authentication](/api-reference/authentication), [Rate Limits](/api-reference/rate-limits), and [Endpoints](/api-reference/endpoint/get-organization-customers).

## OpenAPI

The API reference is available as OpenAPI (Swagger). Use it for code generation or agent tool definitions: [openapi.json](/openapi.json) (or the OpenAPI link in the API reference section).
